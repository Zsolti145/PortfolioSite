Saját fejlesztésű, reszponzív portfólió oldal, amely bemutatja a projektjeimet és webes fejlesztéseimet. Modern, letisztult UI, mobilbarát kialakítás és többoldalas struktúra. A projekt célja a front-end alapok gyakorlása, animációk, elrendezések és komponens-jellegű gondolkodás fejlesztése volt. Deploy Netlify-ra, Git verziókezeléssel : lásd fent.

Használt technológiák: HTML,CSS,Bootstrap,JQuery
